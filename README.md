CodeConnect é uma plataforma social voltada para desenvolvedores, onde os usuários podem compartilhar projetos de programação, adicionar descrições, tags e carregar imagens de seus trabalhos. O objetivo da aplicação é facilitar a troca de conhecimento e exposição de portfólios em uma comunidade voltada para a tecnologia. Este projeto foi desenvolvido com foco em estudos e aprimoramento de habilidades de front-end e JavaScript.

Funcionalidades:
Upload de Imagens: Carregue imagens de seus projetos diretamente na plataforma, utilizando a API FileReader para leitura local de arquivos.
Gerenciamento de Tags: Adicione e remova tags de forma dinâmica aos projetos, categorizando facilmente os trabalhos.
Formulário de Publicação: Preencha o nome, descrição e tags do projeto e publique com facilidade.
Funções Assíncronas: O projeto faz uso de async/await para realizar tarefas assíncronas, como a leitura de arquivos e a verificação de tags, garantindo uma experiência de usuário mais fluida e eficiente.
Experiência Interativa: Interface limpa e intuitiva, com botões de ação como publicar e descartar conteúdo.
Tecnologias Utilizadas:
HTML5 e CSS3 para a estrutura e estilização da aplicação.
JavaScript para adicionar interatividade, como upload de arquivos e gerenciamento de tags.
FileReader API para leitura de arquivos localmente, sem necessidade de upload para o servidor.
Funções Assíncronas (async/await) para gerenciar processos como a verificação de tags e publicação de projetos.
